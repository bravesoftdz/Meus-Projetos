INSTALAÇÃO DO COMPONENTE

ANTES DE INSTALAR:

1) REMOVA completamente a versão demonstração do seu Delphi (caso tenha instalado antes)

2) CERTIFIQUE-SE de que não exista nenhum arquivo referente ao RDprint (DCU ou BPL)  em seu HD, principalmente nos diretórios de instalção do Delphi, nos diretórios de seus projetos ou em qualquer diretórios constantes na lista de PATHS do Delphi.

    * O compilador Delphi, irá anexar o primeiro arquivo que encontrar na lista de Paths, se esta não for a correta, ocorrerá erros na execução do seu aplicativo.
    * Estas dicas valem para qualquer componente que você instalar.

2) LEIA ATENTAMENTE a licença de uso do produto. (click aqui)

INSTALAÇÃO:

O RDprint é enviado e um arquivo ZIP para cada versão do Delphi separadamente:

Exemplo: RDprint40_D7.ZIP --> versão do RDprint 4.0 para Delphi 7


1) Crie um diretório separada para cada versão do Delphi

2) Descompacte o arquivo do RDprint40_D?.ZIP que você recebeu
    * Este pacote deve conter os seguintes arquivos:

2) Estando no Delphi, feche qualquer projeto aberto antes de iniciar a instalação

3) Para instalar siga as opções no MENU do Delphi:

     Component | Install Packges | Add | c:\dir\delprint.bpl

     onde DIR = diretório onde foi descompactado o RDprint

4) Será criado na paleta de componentes a pasta "Deltress" com o componente RDprint

 

APÓS INSTALAR:

1) Se após instalado o componente, ocorrer erro na compilação, dizendo que algum arquivo não foi encontrado, acrescente o PATH
da localizado do componente em

    TOOLS | ENVIRONMENT OPTIONS | LIBRARY | LIBRARY PATH

* Consulte o Manual / Help do Delphi para dúvidas quanto a instalação de Componentes

2) Se ocorrer erro "PROPRIEDADE VALOR INVÁLIDO" ou "PROPRIEDADE NÃO EXISTE", escolha
a opção "IGNORE", abra todos os forms que contêm o RDprint e siga o mesmo procedimento
e só então recompile o projeto.
